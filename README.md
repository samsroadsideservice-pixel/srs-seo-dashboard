# SRS SEO/GEO Dashboard

Public-facing dashboard for Sam's Roadside Service SEO/GEO tracking.

**Live:** https://samsroadsideservice-pixel.github.io/srs-seo-dashboard/

Reads from a published Google Sheet CSV; renders current-state snapshot, rubric score comparisons vs competitors, and weekly ranking deltas.

## Round 1 scope (2026-04-19)
- Services: Mobile Tire / Mobile Battery
- Cities: Indianapolis / Carmel / Fishers
- 6 keyword combinations baselined against state-of-the-art rubric

Data source: Google Sheet (published read-only CSV URL).
Private backup + strategy docs live in a separate private repo.
